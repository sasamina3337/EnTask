# EnTask

EnTaskは時間トラッキングとGoogleカレンダーへのイベントの自動登録を行うWindows Formsアプリケーションです。

## 使い方

1. [リソース](https://github.com/sasamina3337/EnTask)から最新版のEnTaskをcloneします。
2. ダウンロードしたファイルをVisual Studio Cominityでコンパイルし、`EnTask.exe`を実行します。その際に、client_secret.jsonをexeファイルと同じディレクトリ内に配置してください。
3. メインフォームが表示されます。Googleアカウントを用いて認証を行ってください。ここから時間のカウントを開始することができます。
4. カウント開始ボタンを押すと、カウントが開始されます。カウントを停止するには、同じボタンをもう一度押します。
5. カウントを停止すると、カウントした時間がGoogleカレンダーの指定したイベントに自動で登録されます。
6. Todoリスト機能は登録、修正、削除の機能があります。
